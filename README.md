# Domain AI Judge

一个用于「域名估价多 AI 评委打分」的静态网页工具。

## 功能

- 输入域名
- 粘贴域名分析 Markdown
- 内置 AI 评分提示词（6维度加权评分）
- 录入多个 AI 分数
- 自动去掉最高分和最低分
- 自动计算最终均分
- 导出 JSON 结果

## 在线使用

部署到 GitHub Pages 后直接打开即可使用，无需任何后端。

## 玩法说明

1. 输入域名（如 `TEX.COM`）
2. 粘贴你的域名分析 MD 报告
3. 点击「生成完整投喂内容」，复制后分别发给 ChatGPT / Claude / Gemini / DeepSeek / Qwen 等多个 AI
4. 把每个 AI 返回的 `final_score` 填进评委列表
5. 点击「计算结果」，系统自动去掉最高分和最低分，输出最终得分
